⏳ Time Tracker - Gestion du Temps et des Projets
📌 Description
Time Tracker est une application qui permet aux équipes de suivre leur temps de travail et la progression de leurs tâches.

🚀 Fonctionnalités
✅ Gestion des tâches et suivi du temps
✅ Création et assignation des projets
✅ Authentification et gestion des utilisateurs avec Keycloak
✅ Intégration avec MongoDB et Spring Boot

⚙ Technologies Utilisées
Frontend : Angular 16, Bootstrap
Backend : Spring Boot, MongoDB
Authentification : Keycloak
